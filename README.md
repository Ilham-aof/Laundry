# Laundry Management System

A web-based management system designed to help laundry businesses manage orders, customers, and daily operations efficiently. This application provides a dashboard for monitoring business activity, managing order workflows, and tracking customer growth.

---

## 🚀 Features
### 📦 Order Management
- Create new laundry orders
- Update existing orders
- Delete orders
- Track active and completed orders
- View order history

### 👤 Customer Registration & Tracking
- Add new customers
- View total customers
- Monitor new customers per day

### 📊 Dashboard Analytics
- Total number of orders
- Total customers count
- Daily new customer statistics
- Business activity summary

### ✅ Order Completion Workflow
Each order requires two statuses to be completed before being marked as finished:

- **Payment Status** → Completed / Pending
- **Laundry Pickup Status** → Picked Up / Pending

The order will automatically be marked as **Finished** only when both statuses are completed.

### 📄 Pages Included
- Dashboard Page
- Orders Management Page
- History Page
